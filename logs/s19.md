# Week 10 (Dates) - APIs, JSON, Files, and Data Structures

## What I learned this week
This week, I learned how Python can work with real-world data by using APIs, JSON, files, and CSVs. I reviewed core Python concepts like variables, loops, functions, strings, lists, dictionaries, and tuples, then connected them to error handling and data importing/exporting.

I learned that `try/except` helps prevent a program from crashing when something goes wrong, like bad input, a missing key, or a failed request. I also learned that APIs let Python request structured data from a server, and that JSON responses can be converted into Python dictionaries and lists for easier use.

How does an API work?
- We search for data.
- We send a request to a server.
- The server sends back structured data, usually in JSON format.
- Python can read that data and turn it into dictionaries and lists.

A real-time example is the Coindesk API, which sends financial data in a format that is easy for computers to read, but not always easy for humans to read directly.
