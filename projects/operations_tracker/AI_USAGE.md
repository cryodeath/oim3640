# AI Usage

## Initial implementation support

- What I asked: I asked for a starter Flask app and template design for an operations tracker that records tasks, deadlines, budgets, and progress.
- What AI generated: A working Flask application structure with routes for displaying tasks, adding new tasks, and persisting data.
- What I did with it: I used the generated app structure to build `app.py`, created HTML templates in `templates/`, added styling in `static/style.css`, and configured JSON persistence in `data/tasks.json`.
- What I learned: I learned how to structure a Flask project with templates, how to handle form submission with a POST route, and how to keep data persistent using JSON files.
