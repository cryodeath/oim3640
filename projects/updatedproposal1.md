My Project Proposal
What I'm building: A Travel Cost Tracker app that lets me log daily expenses while traveling and see breakdowns by day and category.

Why I chose this: As a Babson student who loves traveling but struggles with budget tracking, I always lose track of what I spend on food vs transport vs activities. This solves my real problem of understanding where my travel money actually goes.

Core features:

Add expenses: date, category (Food/Hotel/Transport/Activities/Other), amount, description

View daily totals: "Mar 15: $287 (3 expenses)"

Category summary: "Food: $309 (45%), Transport: $251 (36%)"

Trip totals and average daily spend

Save/load trips as JSON files

What I don't know yet:

How to make a clean web interface (Streamlit? Flask?)

Best way to organize expense data (dict of dates → list of expenses?)

How to calculate percentages for category pie charts

Whether to add export to CSV/PDF

How to handle multiple currencies if I travel internationally
