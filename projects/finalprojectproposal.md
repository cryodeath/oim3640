# My Project Proposal

## App name:
OpsPulse

## What I'm building:
OpsPulse is an Operations Tracker app that lets student leaders or small teams log tasks, deadlines, budgets, and project progress in one place, then view summaries through a simple dashboard.

## Why I chose this:
As a Babson student involved in operations-heavy leadership and project work, I often have to keep track of many moving parts at once, including deadlines, responsibilities, and budgets. This project solves a real problem for me by organizing scattered operational work into one system that is easier to manage and understand.

## Core features:

- Add tasks: title, owner, due date, category, status, budget, notes

- View all tasks in a dashboard

- Track task status: Not Started, In Progress, Completed, Overdue

- View summary metrics: total tasks, overdue tasks, completion rate, total budget

- Filter tasks by category or status

- Save and load task data using CSV or JSON
